# Thai Story AI App

AI-powered storytelling application with Text-to-Speech

---

## ✨ Features

* User Authentication
* AI Story Generation
* Text-to-Speech Conversion
* Story History Management
* Story Detail Viewing
* Delete Story History

## 🔄 System Flow

User Login/Register
        ↓
Input Story Topic
        ↓
AI Story Generation
        ↓
Text-to-Speech Processing
        ↓
Save Story History
        ↓
Display Story Result

---

## 🏗 System Architecture

Flutter Mobile Application
            ↓
Python Backend API
            ↓
AI Story Generation Service
            ↓
Text-to-Speech Engine
            ↓
Firebase Database
